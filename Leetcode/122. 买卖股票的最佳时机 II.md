## 122. 买卖股票的最佳时机 II
### 解题思路
#### 思路一
****
- 只需要判断是否后面一天的价格比前面一天的高就行了

**python实现代码**
```
class Solution:
    def maxProfit(self, prices):
        """
        :type prices: List[int]
        :rtype: int
        """
        if len(prices) == 0:
            return 0
        return sum([max(prices[i+1]-prices[i], 0) for i in range(len(prices)-1)])

```
