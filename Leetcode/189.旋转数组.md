## 189.旋转数组
### 解题思路
#### 思路一
****
- 可以发现，nums数组中后k位数组会直接顺序不变地排到新数组的前面

**python实现代码**
```
class Solution:
    def rotate(self, nums, k):
        """
        :type nums: List[int]
        :type k: int
        :rtype: void Do not return anything, modify nums in-place instead.
        """
        k = k % len(nums)
        if k != 0:
            tmp = nums[-k:]
            for j in range(len(nums)-1, k-1, -1):
                nums[j] = nums[j-k]
            nums[:k] = tmp


```
