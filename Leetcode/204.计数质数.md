## 204.计数质数
### 解题思路
#### 思路一
****
- 可以看做是’素数筛选法‘。其思想是从小的素数开始，排除该小素数的所有倍数，直到最终剩下的全是素数。  
https://en.wikipedia.org/wiki/Sieve_of_Eratosthenes#Algorithm_complexity

**python实现代码**
```
class Solution(object):
    def countPrimes(self, n):
        """
        :type n: int
        :rtype: int
        """
        isPrime = [1] * n
        i = 2
        while i * i < n:
            if isPrime[i]:
                j = i * i
                while j < n:
                    isPrime[j] = 0
                    j += i
            i += 1
        return sum(isPrime[2:])
```
