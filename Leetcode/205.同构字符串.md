## 205.同构字符串
### 解题思路
#### 思路一
****
- lookup字典用于保存s和t字符的对应关系

**python实现代码**
```
class Solution:
    def isIsomorphic(self, s, t):
        """
        :type s: str6666666666666666t
        :type t: str
        :rtype: bool
        """
        if len(set(s)) != len(set(t)):
            return False
        lookup = {}
        for i, x in enumerate(s):
            if x in lookup:
                if lookup[x] != t[i]:
                    return False
            else:
                lookup[x] = t[i]
        return True

```
