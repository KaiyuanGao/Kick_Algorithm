## 217.存在重复元素
### 解题思路
#### 思路一
****
- 利用字典判断

**python实现代码**
```
class Solution(object):
    def containsDuplicate(self, nums):
        """
        :type nums: List[int]
        :rtype: bool
        """
        lookup = {}
        for num in nums:
            if num in lookup:
                return True
            else:
                lookup[num] = 1
        return False

```
