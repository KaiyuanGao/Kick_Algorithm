## 279.完全平方数
### 解题思路
#### 思路一
**动态规划**
- 如果一个数x可以表示为一个任意数a加上一个平方数bxb，即x=a+bxb， 那么能组成x的最少的平方数的个数也就是能组成a最少的平方数的个数加上1（也就是指的b）

**python实现代码**
```
class Solution:
    def numSquares(self, n):
        """
        :type n: int
        :rtype: int
        """
        if n == 0:
            return 0
        inf = float('inf')
        dp = [inf] * (n+1)
        dp[0] = 0
        dp[1] = 1
        for i in range(2, n+1):
            j = 1
            while j * j <= i:
                dp[i] = min(dp[i], dp[i-j*j]+1)
                j += 1
        return dp[-1]

```
