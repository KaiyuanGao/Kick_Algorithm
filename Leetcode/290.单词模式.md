## 290.单词模式
### 解题思路
#### 思路一
****
- 单词与模式之间的匹配，可以遍历字典来查找出不正确的匹配

**python实现代码**
```
class Solution(object):
    def wordPattern(self, pattern, str):
        """
        :type pattern: str
        :type str: str
        :rtype: bool
        """
        str = str.split(' ')
        if len(pattern) != len(str):
            return False
        lookup = {}
        for i in range(len(str)):
            if pattern[i] not in lookup:
                for key in lookup:
                    if lookup[key] == str[i]:
                        return False
                '''
                上面的遍历考虑以下情况
                    "abba"
                    "dog dog dog dog"
                '''
                lookup[pattern[i]] = str[i]
            elif lookup[pattern[i]] != str[i]:
                return False
                
        return True

```
