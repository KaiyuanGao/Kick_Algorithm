## 最长有效括号
### 解题思路
#### 思路一

**python实现代码**
```
class Solution(object):
    def longestValidParentheses(self, s):
        """
        :type s: str
        :rtype: int
        """
        n = len(s)
        if n == 0:
            return 0
        dp = [0] * n 
        for i in range(n):
            if s[i] == ')':
                left = i-1-dp[i-1]  # 找到i位置对应的左括号
                if left >= 0 and s[left]=="(":
                    dp[i] = dp[i-1] + 2
                    if left > 0:
                        dp[i] += dp[left-1]  #有可能left左边还有子串
        return max(dp)
                    

```
