## 324.摆动排序
### 解题思路
#### 思路一
****
- 首先将原数组排序，然后大的那一半数字降序插在奇数index上，小的那一半数字降序插在偶数index上

**python实现代码**
```
class Solution:
    def wiggleSort(self, nums):
        """
        :type nums: List[int]
        :rtype: void Do not return anything, modify nums in-place instead.
        """
        nums.sort()
        half = len(nums[::2])
        nums[::2], nums[1::2] = nums[:half][::-1], nums[half:][::-1]

```
