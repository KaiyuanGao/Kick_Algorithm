## 整数拆分
### 题目描述
给定一个正整数 n，将其拆分为至少两个正整数的和，并使这些整数的乘积最大化。 返回你可以获得的最大乘积。
### 解题思路
#### 思路一
**dp**
- 和剑指offer上的剪绳子一题一样

**python实现代码**
```
class Solution(object):
    def integerBreak(self, n):
        """
        :type n: int
        :rtype: int
        """
        dp = [i for i in range(n+1)]
        if n == 2:
            return 1
        if n == 3:
            return 2
        for i in range(4, n+1):
            for j in range(1, i):
                dp[i] = max(dp[j]*dp[i-j], dp[i])
        return dp[-1]
        

```
