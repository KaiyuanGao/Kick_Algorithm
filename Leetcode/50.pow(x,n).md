## 50.pow(x,n)
### 解题思路
#### 思路一
**递归**
- 

**python实现代码**
```
class Solution:
    def myPow(self, x, n):
        """
        :type x: float
        :type n: int
        :rtype: float
        """
        if n == 0:
            return 1
        if n < 0:
            return 1 / self.myPow(x, -n)
        if n % 2 == 1:  # n 是奇数
            return x*self.myPow(x*x, n>>1) # 右移运算符，右移n个单位相当于除以2的n次幂，相当于n//2
        else: #偶数
            return self.myPow(x*x, n>>1)
        
```
