## 516 最长回文子序列



### 解题思路
#### 思路一
**dp**
- 其实就是求s和s[::-1]的最长公共子序列

**python实现代码**
```python
class Solution(object):
    def longestPalindromeSubseq(self, s):
        """
        :type s: str
        :rtype: int
        """
        s1 = s[::-1]
        return self.LCS(s, s1)
    
    def LCS(self, s1, s2):
        m, n = len(s1), len(s2)
        dp = [[0]*(m+1) for i in range(n+1)]
        for i in range(1, m+1):
            for j in range(1, n+1):
                if s1[i-1]==s2[j-1]:
                    dp[i][j] = dp[i-1][j-1] +1
                else:
                    dp[i][j] = max(dp[i-1][j], dp[i][j-1])
        return dp[-1][-1]
```
