## 64.最小路径和
### 解题思路
#### 思路一
**动态规划**
- 和第62题的思路类似，只不过这里在每个格子里加上了一个权重

**python实现代码**
```
class Solution:
    def minPathSum(self, grid):
        """
        :type grid: List[List[int]]
        :rtype: int
        """
        if not grid:
            return 0
        m = len(grid)
        n = len(grid[0])
        for i in range(1,m):
            grid[i][0] += grid[i-1][0] 
        for j in range(1, n):
            grid[0][j] += grid[0][j-1]
        for i in range(1,m):
            for j in range(1,n):
                grid[i][j] += min(grid[i-1][j],grid[i][j-1]) 
        return grid[-1][-1]

```
