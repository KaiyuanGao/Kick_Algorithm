## 67.二进制求和
### 解题思路
#### 思路一
****
- 考虑进位的情况

**python实现代码**
```
class Solution(object):
    def addBinary(self, a, b):
        """
        :type a: str
        :type b: str
        :rtype: str
        """
        if a == '' or b == '':
            return a+b
        if a[-1] == '0' and b[-1] == '0':
            return self.addBinary(a[:-1], b[:-1]) + '0'
        elif a[-1] == '1' and b[-1] == '1':
            return self.addBinary(self.addBinary(a[:-1], b[:-1]), '1') + '0'
        else:
            return self.addBinary(a[:-1], b[:-1]) + '1'

```
