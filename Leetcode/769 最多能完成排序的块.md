## 769.最多能完成排序的块
### 解题思路
#### 思路一

> 本题关键在于找出何时应该切断arr？  
观察后可以发现应该是在某一位置前面的所有数字中最大的那个小于后面所有数字中最小的那个，此时可以切块。  
设置一个指针，从左往右遍历arr即可

**python实现代码**
```
class Solution:
    def maxChunksToSorted(self, arr):
        """
        :type arr: List[int]
        :rtype: int
        """
        cnt = 0
        i = 0
        n = len(arr)
        if n ==0: 
            return 0
        if n == 1: 
            return 1
        while i < n:
            if i == n-1:
                cnt += 1
                break
            if max(arr[:i+1]) < min(arr[i+1:]):
                cnt += 1
            i += 1
        return cnt
```
