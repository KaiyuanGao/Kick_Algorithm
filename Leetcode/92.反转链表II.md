## 92.反转链表II
### 解题思路
#### 思路一
****
- 是206题反转链表的变形。所以我们只要先找到需要反转的起始节点和需要反转的区间即可。

**python实现代码**
```
class Solution:
    def reverseBetween(self, head, m, n):
        """
        :type head: ListNode
        :type m: int
        :type n: int
        :rtype: ListNode
        """
        if not head or m == n:
            return head
        p = dummy = ListNode(None)
        p.next = head
        for i in range(m-1):  # 找到需要反转的起始点
            p = p.next
        tail = p.next
        for i in range(n-m):  # 需要反转的区间
            tmp = p.next
            p.next = tail.next
            tail.next = tail.next.next
            p.next.next = tmp
        return dummy.next

```
