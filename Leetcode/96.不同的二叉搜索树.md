## 96.不同的二叉搜索树
### 解题思路
#### 思路一
**DP**
- 从二叉树的根考虑，n个整数中任意一个都可以作为根。当整数k作为根时，它的不同的种类为左子树的种类乘上右子树的种类，即s(k)*s(n-k-1)。所以对于n个整数而言，S(n) = s(0)*s(n-1)+s(1)*s(n-2)+...+s(n-1)*s(0) (n>1)，此外s(0)=s(1)=1
- n=0时，为空树，----> dp[0]=1;  
n=1时， dp[1]=1；  
n=2时，dp[2]=2;  
n>2时，dp[n]=dp[0]*dp[n-1]+dp[1]*dp[n-2]+......+dp[n-1]*dp[0]；

**python实现代码**
```
class Solution:
    def numTrees(self, n):
        """
        :type n: int
        :rtype: int
        """
        dp = [1] * (n+1)
        for i in range(2, n+1):
            s = 0
            for j in range(i):
                s += dp[j] * dp[i-j-1]
            dp[i] = s
        return dp[-1]
        
        

```
