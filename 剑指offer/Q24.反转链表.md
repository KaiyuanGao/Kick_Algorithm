## Q24.反转链表
### 题目描述
输入一个链表，反转链表后，输出新链表的表头。
### 解题思路
#### 思路一
****
- 注意链表的转换，直接看代码配合小例子分析比较容易理解

**python实现代码**
```python
class Solution:
    # 返回ListNode
    def ReverseList(self, head):
        # write code here
        if not head:
            return None
        prev = None
        cur = head
        while cur:
            nxt = cur.next
            cur.next = prev
            prev = cur
            cur = nxt
        return prev
```
