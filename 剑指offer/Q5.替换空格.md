## 替换空格
### 题目描述
请实现一个函数，将一个字符串中的每个空格替换成“%20”。例如，当字符串为We Are Happy.则经过替换之后的字符串为We%20Are%20Happy。
### 解题思路
#### 思路一
****
- 如果你用的C++的话，在替换的时候就需要考虑内存问题，因为空格只占有一个内存空间，而%20占了三个内存空间，所以需要考虑在替换的同时不会把该字符串后面的内存覆盖修改。
- 对于python来说，一行解决

**python实现代码**
```python
class Solution:
    # s 源字符串
    def replaceSpace(self, s):
        # write code here
        return s.replace(' ', '%20')
```
